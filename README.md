<h1><b>Azure-Sentinel-SIEM-Honeypot-for-Real-Time-RDP-Attack-Monitoring-and-Geolocation-Visualization</b></h1>
This project demonstrates a cloud-based cybersecurity monitoring system using Microsoft Azure Sentinel integrated with a honeypot virtual machine. The VM is intentionally exposed to capture real-time RDP brute force attacks from across the world. Security logs are collected in Log Analytics Workspace and enriched with geolocation data using a custom PowerShell script and API. The processed data is visualized through interactive dashboards, including a global attack map, to analyze threat patterns and attacker origins. This project highlights practical SIEM implementation, log analysis, and threat intelligence integration, making it valuable for learning real-world cybersecurity monitoring and incident detection techniques.
