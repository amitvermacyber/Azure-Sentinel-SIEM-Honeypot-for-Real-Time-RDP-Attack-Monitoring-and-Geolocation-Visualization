# Azure-Sentinel-SIEM-Honeypot-for-Real-Time-RDP-Attack-Monitoring-and-Geolocation-Visualization
Cloud-based SIEM project using Microsoft Azure Sentinel and a honeypot VM to capture real-time RDP brute force attacks. Logs are enriched with geolocation data via PowerShell and visualized on a global dashboard, providing insights into attack patterns and threat intelligence.
